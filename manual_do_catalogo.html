1. Visão Geral

O CineCatalog Elo é um sistema completo de gestão de acervo cinematográfico. Permite cadastrar filmes, séries e estreias, organizar por géneros, avaliar com estrelas, favoritar, criar lembretes, gerar listas A4 e muito mais.

  • Todo o código está num único ficheiro index.html
  • Os dados são armazenados no localStorage do navegador
  • Suporta exportação/importação JSON para backup
  • Funciona em navegadores desktop, Electron e Smart TV
  • Auto salvamento para ficheiro na pasta do acervo configurada

2. Abas (Filmes / Séries / Estreias)

No topo do header encontra três abas de navegação principal:

  • Filmes — Lista todos os filmes cadastrados
  • Séries — Lista todas as séries cadastradas
  • Estreias — Lista estreias futuras e passadas

Ao clicar numa aba, o conteúdo do acervo é filtrado automaticamente.

3. Cadastro de Filmes/Séries

Clique no botão CADASTRAR no header para abrir o modal de cadastro.

  • Séries: Título, Ano, Duração, Temporada, Episódio, Diretor, Elenco, Género, Sinopse, Capa, Trailer, Media, Classificação, Status
  • Estreias: Data, Título, Tipo (Filme/Série), Trailer URL. Suporta múltiplas estreias dinâmicas com adição/remoção em tempo real
  • Preenchimento automático dos campos de caminho (Cards, Vídeos) conforme configurado

5. Pesquisa com Destaque Neon

Clique no ícone (ou pressione Ctrl+F) para abrir a barra de pesquisa.

  • Pesquisa por título, diretor, elenco, género, sinopse
  • Mínimo de 3 caracteres
  • Resultados destacados com neon laranja
  • Suporta pesquisa por voz (navegadores com SpeechRecognition)

Pesquisa Booleana

Use o operador + entre termos para combinar múltiplos critérios. O sistema filtra itens que correspondem a todos os termos.

Exemplos:

ExemploO que faz
açãoEncontra todos os filmes/séries com "ação" no título, género ou sinopse
ação+2023Filmes de ação do ano 2023
terror+2024+alemãoFilmes de terror alemães de 2024
star+warsBusca exata por "star wars" (ambos os termos presentes)
comédia+romance+2022Comédias românticas de 2022
nolan+ficçãoFilmes do diretor Christopher Nolan no género ficção científica

10. Primeiros Passos — Configuração Inicial

Para começar a usar o CineCatalog Elo corretamente, siga estas recomendações na ordem indicada.

10.1 Configurar os Caminhos (Pastas do Acervo)

Abra as Configurações (ícone no header) e localize a secção Caminhos.

  • Cards: Pasta onde serão guardadas as imagens de capa dos filmes/séries (ex: D:\ACERVO\CARDS)
  • Vídeos: Pasta onde estão armazenados os ficheiros de mídia (ex: D:\ACERVO\VIDEOS)
  • Backups: Pasta para exportar e importar o ficheiro JSON de backup
  • Acervo Geral: Define o nome do ficheiro (ex: meu_acervo.json) e a pasta onde o auto salvamento guardará os dados

Use o botão 📁 ao lado de cada campo para abrir o seletor de pastas. No Electron, abre o diálogo nativo do sistema; no navegador, usa a File System Access API.

Importante: Os caminhos devem ser absolutos (ex: C:\Users\...\ACERVO\CARDS) e as pastas devem existir no disco.

Preenchimento Automático: Quando os caminhos Cards e Vídeos estão configurados, o sistema tenta preencher automaticamente os campos "Capa" e "Mídia" no formulário de cadastro, usando como base o título do filme/série.

10.2 Configurar o Auto Salvamento

Na mesma secção de Configurações:

  1. Defina um nome para o ficheiro do acervo (ex: acervo.json)
  2. Escolha a pasta de destino onde o ficheiro será salvo
  3. Ative o toggle "Auto Salvamento"

Quando ativo, o sistema guarda automaticamente o estado completo do acervo no ficheiro sempre que houver alterações (adicionar, editar, remover itens, favoritar, etc.).

No rodapé, um indicador com ícone de disquete mostra o estado:

  • Azul neon = Auto salvamento ativo
  • Cinza = Auto salvamento desligado

10.3 Exportar / Importar JSON (Backup)

O sistema permite exportar e importar todo o acervo num único ficheiro JSON.

Exportar: Clique no ícone no header. O navegador descarrega um ficheiro cinecatalog_backup_YYYY-MM-DD_HHmmss.json com todos os dados.

Importar: Clique no ícone no header e selecione um ficheiro JSON válido. O sistema substitui o acervo atual pelos dados importados.

10.4 O Ficheiro JSON — O que guarda e qual a sua importância

O ficheiro JSON é a base de dados completa do CineCatalog Elo. Ele contém:

  • Todos os filmes, séries e estreias cadastrados (com título, ano, género, sinopse, elenco, etc.)
  • Status de cada item: Novo, Assistir, Favorito
  • Avaliações por estrelas (0–5)
  • Lembretes e notificações associados a cada item
  • Episódios de séries (títulos, durações, links)
  • Datas de criação e IDs únicos
  • Preferências de géneros personalizados

Importância: Este ficheiro é a cópia de segurança do seu acervo. Se o localStorage do navegador for limpo ou corrompido, pode restaurar todos os dados importando este JSON. Guarde-o num local seguro (disco externo, nuvem) e faça exportações periódicas.

O JSON segue o formato padrão e pode ser aberto em qualquer editor de texto para consulta ou edição manual avançada.

10.5 Verificação após Configuração

  • Confirme que o indicador de auto salvamento no rodapé está azul
  • Faça um teste: cadastre um filme, depois exporte o JSON e verifique se o ficheiro contém os dados
  • Importe o ficheiro exportado para um perfil/navegador diferente para confirmar que a restauração funciona

15. Executar Mídia (playMedia)

O sistema usa a função Logic.playMedia(id) para reproduzir vídeos:

  • No Electron: usa electronAPI.openMedia() ou electronAPI.openWithPlayer() com o player selecionado (VLC, MPC-HC, etc.)
  • Players abrem em tela cheia/maximizado automaticamente
  • No navegador: fallback para window.open()
  • Trailers abrem no navegador padrão em tela cheia
  • Os botões "EXECUTAR FILME" e "EXECUTAR EPISÓDIO" nos modais INFO usam este sistema
  • O botão "ASSISTIR" na Sugestão do Dia também usa playMedia()

Dados não carregamVerifique se o localStorage não está corrompido. Use "Importar" para restaurar um backup

19. Segurança

  • Teclas bloqueadas: F12, F11, F5, Ctrl+Shift+I, Ctrl+Shift+J, Ctrl+Shift+C, Ctrl+U — impedem acesso a DevTools
  • Context menu: Clique direito desabilitado em toda a aplicação (exceto menu personalizado)
  • Anti-frame: A página não pode ser carregada dentro de iframes (clickjacking)
  • Console: Desabilitado em produção (não localhost/file)
  • DevTools hook: Detectores de DevTools neutralizados
  • Fechamento: Aviso ao fechar com modal de cadastro aberto

24. Estatísticas com Filtros em Tempo Real

O dashboard de Estatísticas do Acervo agora inclui um sistema completo de filtros temporais que permite analisar cadastros e ações do utilizador por diferentes períodos:

Tipos de Filtro Disponíveis

FiltroDescrição
DiaMostra apenas os dados de um dia específico. Use o seletor de data para escolher o dia.
Período de DiasSelecione um intervalo de dias (data início e data fim) para ver os dados desse período.
MêsFiltra por um mês específico do ano. Escolha o mês e o ano no seletor.
Período de MesesSelecione um intervalo de meses (mês início e mês fim) dentro de um ano.
Ano InteiroMostra todos os 12 meses do ano selecionado, com distribuição mensal visível.
Período de AnosSelecione um intervalo de anos (ano início e ano fim) para visão abrangente.

Os filtros são aplicados em tempo real — ao alterar qualquer critério, os gráficos e cards de estatísticas atualizam instantaneamente sem necessidade de recarregar.

Dica: Use os filtros para comparar períodos específicos, identificar tendências de cadastro e acompanhar o crescimento do acervo ao longo do tempo.
